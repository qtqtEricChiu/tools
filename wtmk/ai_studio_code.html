效果说明

合成效果为顶层放置目标图片，底层放置背景图片，目标图片上、左、右三边与背景图片贴齐。

高斯模糊背景效果

启用高斯模糊后，将创建四层结构：

  1. 高斯模糊层：目标图片模糊后作为底层背景
  2. 颜色遮罩层：半透明颜色层覆盖在模糊层上
  3. 背景图片层：用户选择的背景图片
  4. 目标图片层：原始目标图片置于顶层
图片处理流程
  1. 检测图片方向：横向或竖向
  2. 竖向图片：先向右旋转90度变为横向（尺寸为3840×2160）
  3. 小尺寸图片：拉伸至3840×2160
  4. 合成图片：背景在下层，目标图片在顶层
  5. 添加文字水印：根据设置的文字样式和位置
  6. 竖向图片：将合成结果向左旋转90度恢复原始方向
注意事项
  • 处理高分辨率图片需要时间，请耐心等待
  • 竖版图片处理时间较长（需要两次旋转操作）
  • 请确保选择的背景图片尺寸为3840×2707
  • 建议每次处理图片不超过20张，避免浏览器卡顿
