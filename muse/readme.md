### What is this
完全支持**纯手柄/键盘**操作的本地音乐播放器。

完全本地，暂时没有引入在线歌曲信息等资源的想法。

适配QQ音乐早期支持下载的ogg、mp3、flac格式文件，包括qmcflac格式文件（需要解密转换，此处不提供相关服务），支持识别内嵌信息

适配QQ音乐下载附带的lrc文件

支持Chrome Media Session API，并通过其向Android系统锁屏等场景提供相关服务

支持沉浸模式

手柄支持任意xbox手柄
